# Security Policy

## Supported versions

Only the latest release on the `main` branch is supported.

## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Please report them privately via GitHub:
[Open a private vulnerability report](https://github.com/foXaCe/alexa_media_player/security/advisories/new).

Target response time: within 7 days.

Because this integration relies on an unofficial Amazon API, please also avoid posting
account credentials, cookies or tokens in any public channel.
