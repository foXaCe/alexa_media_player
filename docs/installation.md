# Installation

## HACS (recommended)

1. In HACS, add `https://github.com/foXaCe/alexa_media_player` as a custom repository
   (type: **Integration**).
2. Search for **Alexa Media Player** and install it.
3. Restart Home Assistant.
4. Go to **Settings → Devices & Services → Add Integration → Alexa Media Player**.

## Manual

1. Copy `custom_components/alexa_media/` into `<config>/custom_components/`.
2. Restart Home Assistant.
3. Add the integration from the UI.

See the [wiki](https://github.com/foXaCe/alexa_media_player/wiki) for detailed, up-to-date instructions.
