# Configuration

Configuration is done entirely through the UI:

**Settings → Devices & Services → Add Integration → Alexa Media Player**

You will be asked for your Amazon credentials and (when required) to complete a
captcha / OTP flow. Options such as scan interval, included/excluded devices and
extended entity discovery can be changed later via the integration's **Configure** button.

For the full list of options and advanced setups, see the
[Configuration wiki page](https://github.com/foXaCe/alexa_media_player/wiki/Configuration).
